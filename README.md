# Memory Game

Jeu de mémoire développé avec : MongoDB, Express, React, Node.js.

## Instructions de Lancement

Assurez-vous d'avoir installé et lancé :
* **Node.js**
* **MongoDB** (Le serveur doit tourner localement sur le port `27017`).

### 2. Installation et Démarrage

Il faut lancer le Backend et le Frontend dans deux terminaux séparés.

#### Étape 1 : Lancer le Serveur (Backend)
Ouvrez un terminal et tapez :

cd backend
npm install
npm start

#### Étape 1 : Lancer le Serveur (Backend)

ouvrez un deuxieme terminal et tapez : 
cd frontend
npm install
npm start
